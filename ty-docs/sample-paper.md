## First Level Subheading

This paragraph follows a first-level subheading, which should be centered and bold with a triple space above it and a double space below. The following block quotation should be single-spaced and indented half an inch with no quotation marks:

> This is a demonstration block quotation standing in for a passage of five or more lines. It should render single-spaced and indented as a block, set apart from the surrounding double-spaced text, with a footnote at the end to cite its source.[^visual]

The paragraph after the block quote returns to normal body formatting.
